## 概要

このテンプレートはデフォルトでパブリック API を提供しません。このページを、追加するエンドポイントのドキュメント作成の出発点としてご利用ください。

## ベース URL

サービスがデプロイされたら設定してください。例：

```
https://api.example.com
```

## 認証

必要なヘッダーやトークンをここに記述してください（例：Bearer トークンや API キー）。

## エラーフォーマット

API が返すエラーの形式を文書化してください。例：

```
{
  "error": "string",
  "message": "string"
}
```

実装されたエンドポイントをここに追加してください。
